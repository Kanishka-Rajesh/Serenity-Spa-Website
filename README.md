# 📖 Overview

Serenity Spa Website is a responsive frontend web application designed for a luxury spa and wellness center.

The project focuses on delivering an elegant user experience through modern layouts, responsive design, and interactive web pages using only HTML, CSS, and JavaScript.

It demonstrates frontend web development concepts without requiring any backend server or database.
